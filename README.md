🛡️ Secure Telegram OTP API

Next-Gen Authentication Gateway

A high-performance, silent-mode Flask API designed for secure Two-Factor Authentication (2FA). This system dispatches encrypted one-time passwords directly to a Telegram Bot, featuring a sleek, modern Glassmorphism interface.

💎 Premium Features

🕵️ Invisible Logging: Customized Werkzeug configuration to suppress all networking/request logs in the console.

🎨 Pro UI: Stunning dark-mode dashboard with "Plus Jakarta Sans" typography.

🔒 Security First: Full .env support for sensitive Bot Tokens and Chat IDs.

⚡ Real-time UX: Dynamic status updates and countdown logic.

🌍 Cross-Origin Ready: Pre-configured CORS for seamless frontend-backend communication.

🛠️ Tech Stack

Backend: Python / Flask

Frontend: HTML5 / CSS3 (Glassmorphism) / Vanilla JS

Integration: Telegram Bot API

Environment: Dotenv / Flask-CORS

🚀 Quick Start Guide

1. Installation

Clone the repository and install the dependencies:

git clone [(https://github.com/blaxkmiradev/Secure-Telegram-OTP-API.git](https://github.com/blaxkmiradev/Secure-Telegram-OTP-API.git)]
cd Secure-Telegram-OTP-API
pip install -r requirements.txt


2. Configuration

Create a .env file in the root folder and add your credentials:

BOT_TOKEN=your_bot_token_here
CHAT_ID=your_chat_id_here


3. Execution

Launch the silent API:

python api.py


Open index.html in your browser to access the portal.
